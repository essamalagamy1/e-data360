# EDATA 360 - موقع تحليل البيانات الاحترافي

## نظرة عامة

موقع عصري وحديث لشركة EDATA 360 المتخصصة في تحليل البيانات وإنشاء لوحات التحكم الاحترافية باستخدام Excel و Power BI.

## التقنيات المستخدمة

-   **Laravel** - إطار عمل PHP
-   **Tailwind CSS** - للتصميم والأنيميشن
-   **Font Awesome** - للأيقونات
-   **Google Fonts** - خطوط Cairo و Inter

## الصفحات المحدثة

## الأنيميشن والتأثيرات

-   **Pulse Animation**: للخلفيات المتحركة
-   **Hover Effects**: Scale, Translate, Color transitions
-   **Gradient Animations**: للنصوص والأزرار
-   **Shadow Effects**: للبطاقات والعناصر التفاعلية

## التثبيت والتشغيل

### 1. تثبيت المتطلبات

```bash
composer install
npm install
```

### 2. إعداد البيئة

```bash
cp .env.example .env
php artisan key:generate
```

### 3. إعداد قاعدة البيانات

```bash
php artisan migrate
php artisan db:seed
```

### 4. تشغيل المشروع

```bash
# في terminal منفصل
npm run dev

# في terminal آخر
php artisan serve
```
