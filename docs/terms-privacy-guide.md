# تعديل الشروط والأحكام وسياسة الخصوصية

## من الداشبورد (Filament)

1. افتح الداشبورد على `/admin`
2. اذهب إلى **إعدادات الشركة** (Company Settings)
3. اضغط على زر التعديل للسجل الموجود
4. ستجد 4 أقسام قابلة للطي:
    - **معلومات الشركة**: معلومات الاتصال والبيانات الأساسية
    - **الشعارات**: شعار الشركة والفافيكون
    - **الشروط والأحكام**: محتوى صفحة الشروط والأحكام
    - **سياسة الخصوصية**: محتوى صفحة سياسة الخصوصية

## صيغة البيانات (JSON)

البيانات مخزنة بصيغة JSON بالشكل التالي:

```json
{
    "intro": "نص المقدمة",
    "sections": [
        {
            "title": "عنوان القسم الأول",
            "content": "محتوى القسم"
        },
        {
            "title": "عنوان القسم الثاني",
            "content": "وصف القسم",
            "list": ["بند أول", "بند ثاني", "بند ثالث"]
        }
    ]
}
```

### الحقول المتاحة:

-   **intro**: نص المقدمة (إجباري)
-   **sections**: مصفوفة من الأقسام (إجباري)
    -   **title**: عنوان القسم (إجباري)
    -   **content**: محتوى القسم (اختياري)
    -   **list**: قائمة نقطية (اختياري)

## مثال كامل للشروط والأحكام:

```json
{
    "intro": "باستخدامك لخدمات E-DATA 360، فإنك توافق على الالتزام بهذه الشروط والأحكام. يرجى قراءتها بعناية.",
    "sections": [
        {
            "title": "الخدمات",
            "content": "نقدم خدمات تطوير مواقع الويب، تطبيقات الجوال، تصميم واجهات المستخدم، والحلول البرمجية المتكاملة وفقاً للمواصفات المتفق عليها مع العميل."
        },
        {
            "title": "الدفع",
            "content": "شروط الدفع:",
            "list": [
                "يتم دفع 30% مقدماً قبل بدء العمل",
                "يتم دفع 50% اثناء العمل ومتابعة العميل في تقدم وقبل عملية تسليم المشروع",
                "يتم دفع 20% عند تسليم المشروع بشهر كامل من تاريخ التسليم لضمان ثقة العميل"
            ]
        }
    ]
}
```

## ملاحظات مهمة:

1. **تأكد من صحة صيغة JSON**: استخدم أداة مثل [JSONLint](https://jsonlint.com/) للتحقق من صحة الصيغة قبل الحفظ
2. **النص بالعربية**: تأكد من استخدام النصوص العربية بشكل صحيح
3. **الأقسام تظهر بالترتيب**: الأقسام ستظهر في الموقع بنفس ترتيبها في المصفوفة
4. **الترقيم تلقائي**: الأرقام (1، 2، 3...) تضاف تلقائياً للأقسام

## عرض الصفحات:

-   **الشروط والأحكام**: `/terms`
-   **سياسة الخصوصية**: `/privacy`
